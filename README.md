# TravelTrucks - Frontend

Kısa açıklama: TravelTrucks için Vite + React ile hazırlanmış frontend starter.

## Nasıl çalıştırılır

1. `npm install`
2. `npm run dev`

## Özellikler

- Katalog listeleme
- Filtreleme (backend query param ile)
- Favorilere ekleme (localStorage)
- Karavan detay sayfası ve rezervasyon formu (toast bildirim)
